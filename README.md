# Smart Choice ฉลาดเลือก - Static Website

เว็บ HTML/CSS/JS สำหรับลง GitHub Pages ได้ทันที

## วิธีใช้งาน
1. อัปโหลดไฟล์ทั้งหมดขึ้น GitHub Repository
2. ไปที่ Settings > Pages
3. เลือก Deploy from a branch
4. Branch: main / Folder: root
5. กด Save

## แก้ไขที่สำคัญ
- เปลี่ยนอีเมลใน `index.html`
- ใส่ YouTube Embed แทนคำว่า `YouTube Embed Area`
- ใส่ Affiliate Link ในปุ่มสินค้า
- เพิ่มบทความจริงตามหมวดหมู่
